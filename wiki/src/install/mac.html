Tails does not work on Mac models that use the Apple M1 chip.

Mac models that use the M1 chip include:

  • iMac (24-inch, M1, 2021)
  • MacBook Pro (13-inch, M1, 2020)
  • MacBook Air (M1, 2020)
  • Mac mini (M1, 2020)

It is a [[known issue|support/known_issues#mac]] that with recent MacBook Pro models, the keyboard, trackpad, and Touch Bar might not work.

As a workaround, you can use an external keyboard and mouse.

For more information about this bug, see [[!tails_ticket 13576]].
